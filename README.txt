Entrega Parcial 1 - Sistemas Distribuídos

O Sistema de aviso de ocorrências de interrupção de tráfego em rodovias (SAOITR) é um programa cliente-servidor, que realiza a troca de mensagens através de sockets.
Na primeira entrega desse projeto, estão incluídas as funcionalidades de Login, Cadastro e Logout, por enquanto sem uso de interface gráfica e banco de dados.
Para executar tanto o servidor quanto o cliente, é possível utilizar uma IDE para Java, como o Eclipse, NetBeans ou VS Code, ou fazer a execução do arquivo pelo prompt de comando. Ex: javac EchoClient2.java
Esse projeto utiliza apenas uma biblioteca externa, o Gson, que padroniza a troca de mensagens entre cliente e servidor. O arquivo JAR do Gson já está incluído nas bibliotecas deste projeto.
